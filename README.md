# 📖 About

**SentinelSOC** is an open-source, modular Security Operations Center (SOC) toolkit designed for **Linux** and **Termux**.

The goal of this project is to provide cybersecurity students, Blue Teamers, and SOC analysts with a portable command-line toolkit that combines multiple defensive security utilities into one application.

Instead of using several standalone tools, SentinelSOC brings together network analysis, threat intelligence, log analysis, digital forensics, and reporting in a clean, modular interface.
